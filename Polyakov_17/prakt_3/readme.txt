Задание 2

Используя шаблон Proxy, определить заместитель для NumPrinter, обладающий следующими свойствами:

длительность работы метода int[] GetNumbers() не должна превышать одну секунду;
метод int[] GetNumbers() должен возвращать массив нечетных чисел.
Объект-заместитель должен содержать ссылку на замещаемый объект и использовать его методы.
